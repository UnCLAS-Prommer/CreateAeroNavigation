# 机械动力：航空学导航
本模组是“机械动力：航空学”的附属模组“机械动力：航空学导航”

模组添加了VOR导航台，DME测距台，路径点记录图（暂定名称），为原版航空学的导航提供了更多可能。

使用方法很简单，只需要将路径点记录图放到航空学的导航台，在到达目标点时导航台会向下方发出一个红石脉冲，并自动将导航台目的地设置为下一个目标点。
